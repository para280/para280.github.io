  • Go to Anabas Conference Management System login page.
  • Click "Account Registration" button. to get the account register page.
  • If you already registered, type your user ID and password.

  • After fill out all the required fields, click "Register" Button.
  • This page will appear only for registration.


  • If you are successfully logged in, you will see this welcome page.
  • This page is your homepage that will display meetings that you are hosting and invited.
  • You can go to this home at anytime by clicking "Home" button in the menu bar at the left side.
  • Click "Schedule Meeting" button to create a new meeting schedule.
  • Click "Configure Software" to checking your computer and installing the Anabas conference application.


  • If you already have meetings that you are hosting or invited, they will be displayed under the heading "You Are Hosting These Meetings" or "You Are Invited To These Meetings" in your home page.


  • If you Click "Configure Software" in the left menu bar to configure and install the application, this window will pop-up.
  • Click "Yes" to start configure software.
  • If you are a first time user, you should configure software on your computer at least 30 minutes before the first meeting.


  • If you click "yes", this window will pop-up, and configure and install software will automatically start working.
  • After software is successfully installed, the page will automatically go to the "Configure Audio" page to configure audio on your machine.


  • This audio configuration allows you to test if your speaker and microphone is compatible with the Anabas software, and you can set up the microphone and speaker volume.
