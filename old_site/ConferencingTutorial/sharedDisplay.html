  • If you click anywhere on the shared display title tool bar,shared display tool bar will be expanded, and grey shared display window will be appear (white board will be hidden).
  • Shared display window is originally grey, and this is to load a documents, files, url or applications to share the view in the meeting.
  • At the top of the shared display window, there is the address tool bar to indicate the address of resources.
  • You can type an address to get URL or files from your local computer.
  • Only host can browse documents, files or URLs to share.
  • To upload a file from the local directory in your computer to share, click at the top of the application window, then you can open files to upload and share.
  • If the document/file that you want to share is not in your computer, download the document/file by clicking at the left top of the appliaction window.

  • Type document location (URL or file location in your local computer) in the address box to get a URL or a file.
  • You can go to previous page or next page by clicking or at the top.
  • Please use the top box and buttons to get URL or go to previous or next, instead of using buttons or box in the shared browser.
  • Only host can use the tools under shared disply tool bar.
  • Tools in Participant's view will be inactive and displayed with grey.

  • Tools Description:
    - click to choose sharing application from list (only currently running appliactions list which normally appears in task bar on your monitor will appear).
    - click to pause sharing, to resume, click again.
    - highlight on the shared display window.
    - make red arrow mark on the shared display window by clicking where you want to mark.
    - erase all marks or highlights on shared display window.
